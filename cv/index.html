Final Project: Enhanced Augmented Reality

For this project, students were required to create a simple markerless augmented reality system, similar to the one below, but without the corner markers. To accomplish this task, I used an image tracker system which uses a combination of image recognition to identify a target and then search through input frames for positive identification.

Each frame, a number of steps are completed including feature extraction and matching, homograpy estimation and refinement, and pose estimation based on a known camera calibration.
